SSShow Example page

Shameen's SlideShow (SSShow) is a JavaScript/jQuery-powered slideshow. It resizes images to fit its container, and makes them center-aligned. There's a simple fade effect to show and hide each image.
Here's another example with multiple instances on one page.
The following code sets up the SShow example: 

<script type="text/javascript" src="https://ajax.googleapis.com/ajax/libs/jquery/1.5.0/jquery.min.js"></script>
<script type="text/javascript" src="ssshow.js"></script>
<script type="text/javascript">
function SSShow_doExample() {
    //Array of images with source, width and height (in pixels)
    var imgs = [];
    imgs[0] = ["images/img001.jpg", 400, 299];
    imgs[1]=["images/img002.jpg", 328, 420];
    imgs[2]=["images/img003.jpg", 275, 208];
    
    var delay = 3000;
    var margin = 5;
    
    //Start the slideshow
    SSShow_start('#SShowID',imgs,delay,margin);
}
</script>

Let's go through this code step-by-step.
  1. var SSS1 = {};
    • This creates an object that will keep the properties that we define.
    • It doesn't really matter what you call it.
  2. imgs = [];
    • This is an array that stores the data on the images you want to be displayed.
    • The slideshow will start at [0] and loop through the rest.
    • Each item is also an array, storing the source URL, width, and height.
    • Image dimensions must be specified
  3. margin and delay
    • margin is the minimum gap between the image and its container.
    • delay is the time in milliseconds to wait before showing the next image.
  4. SSShow_start('#SShowID',SSS1);
    • This will begin the slideshow.
    • The first parameter is the target/container for the slideshow (in this case, id="SShowID"), and SSS1 is the object we just made.
